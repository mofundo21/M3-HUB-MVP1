Exact contents:
```jsx
import React, { createContext, useContext } from 'react';
import { useDeviceDetection } from '../hooks/useDeviceDetection';

export const DeviceContext = createContext({
  isMobile: false,
  isTablet: false,
  isDesktop: true,
});

export function DeviceProvider({ children }) {
  const value = useDeviceDetection();
  return <DeviceContext.Provider value={value}>{children}</DeviceContext.Provider>;
}

export function useDevice() {
  return useContext(DeviceContext);
}
```

Rationale (per RESEARCH Pattern 2): Context fanout avoids prop drilling through Hub3D → MobileControls / future HUD / LoginForm consumers (5+ planned). Default value mirrors the SSR-safe shape from Task 1.
